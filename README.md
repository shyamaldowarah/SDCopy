## Installation

### Installer (Recommended)
1. Download **`SDCopy_Setup.exe`** from the [Releases](../../releases) page.
2. Run the installer and follow the prompts.
3. SDCopy will be added to your Start Menu.

### Portable
1. Download **`SDCopy_Portable.zip`** from the [Releases](../../releases) page.
2. Extract the zip file to any folder.
3. Run `SDCopy.exe`.

## Usage

- **Copy**: Click the "Copy" button on any card to copy text back to your clipboard.
- **Edit**: Click the pencil icon or double-click a card to edit its content.
- **Delete**: Click the trash icon to remove an item.
- **Shortcuts**:
    - `Ctrl + S`: Quick Save / Add New Note.
